# 프로젝트 정책 및 헌법 (Project Constitution)

## 1. 언어 정책 (Language Policy)
- **전용 언어**: 한국어 (Korean)
- **적용 범위**: 모든 대화, 주석, 문서(README, task.md 등)는 반드시 **한국어**로 작성한다.

## 2. 작업 원칙 (Working Principles)
- **사용자 주도 (User-Driven)**: 
    - AI는 사용자의 명시적인 지시 없이 독단적으로 판단하거나 "앞서 나가는" 행위를 절대 금지한다.
    - 이전 프로젝트의 실패 경험이나 외부 지식을 함부로 적용하지 않는다. 오직 사용자가 제공하는 정보만 신뢰하고 실행한다.
- **기술 스택**:
    - **Core**: `curl-cffi`
    - **Target**: Coupang APK 기반 자동화

## 3. 개발 워크플로우
1. 사용자가 지식/정보 제공
2. AI는 해당 내용에 기반한 최소한의 구조/코드 작성
3. 사용자 검토 및 승인
4. 다음 단계 진행
